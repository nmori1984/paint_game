# instructions for copilot
静的なhtmlファイルおよびjsファイルによってWEBサイトを作ります。
サイトの中身はお絵描きゲームのための、①描画キャンバス、②お題のルーレット、③カウントダウンタイマー、です。

## ①描画キャンパス
- スマホの横長画面で表示
- 画面の左側60%分を使用
- 白地の背景。色パレット、線の太さ３択。指でスマホ画面をなぞって描画。消しゴムツール、REDO、UNDO、全クリアボタン。
- ダウンロードボタンをタップするとJPEG形式でローカルにDLする
## ②お題のルーレット
- お絵描きのお題を決めるルーレット。ランダムに選択できる形式であればなんでもよい。
- roulette.txtにお題のリストを記録。
## ③カウントダウンタイマー
- 開始ボタン
- 停止ボタン
- 初期化ボタン：30s, 60s, 100s


## 大まかな構成はOKです！細かい修正をいくつかお願いします。
- ユーザは幼稚園児です。漢字の使用はNGです。
- タイマーの表示をミリ秒単位にしてください

## おだいルーレットを画面上で編集できるようにしてください。
編集は大人が実行するので、幼稚園児向けの考慮は不要です。

- おだいルーレットはお題を編集ボタンが半分くらいで見切れました。
- タイマーは完全に見切れてしまいました。
