# Paint Game

A simple web-based paint application built with HTML5 Canvas, CSS, and JavaScript.

## Features

- **Drawing Canvas**: Draw freely on an 800x600 pixel canvas
- **Color Picker**: Choose any color for your brush
- **Brush Size Control**: Adjust brush size from 1 to 50 pixels
- **Clear Canvas**: Clear the entire canvas with one click
- **Touch Support**: Works on mobile devices with touch support

## How to Use

1. Open `index.html` in a web browser
2. Select a color using the color picker
3. Adjust the brush size using the slider
4. Click and drag on the canvas to draw
5. Click "Clear Canvas" to start over

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Canvas drawing functionality

## Technologies Used

- HTML5 Canvas API
- CSS3
- Vanilla JavaScript (ES6+)
